#### Setup Instructions

###### Prerequisites
- Node + npm
- React

###### Development Setup
- Clone the Repository
- Setup Backend-Server
 <code>npm install</code>
 <code>npm start</code>


- Setup React-Server
 <code>cd client</code>
 <code>npm install</code>
 <code>npm start</code>

- Open-Browser
 <code>http://localhost:3000</code>
